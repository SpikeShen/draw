# Draw.io 架构图集合

本目录用于存放通过 **Kiro + Draw.io MCP** 生成的各种架构图。

## 关于

这些架构图是利用 Kiro AI 助手结合 Draw.io MCP Server 自动生成的，展示了不同系统的技术架构设计。通过 AI 驱动的方式，可以快速将架构设计思路转化为专业的可视化图表。

## 特点

- 🤖 **AI 驱动生成**：使用 Kiro AI 自动创建和编辑架构图
- 🎨 **AWS 官方图标**：使用 AWS 官方图标库，确保专业性
- 🔄 **快速迭代**：通过对话即可修改和优化架构设计
- 📐 **标准化输出**：统一的绘图风格和规范

## 技术栈

- [Kiro](https://kiro.ai) - AI 编程助手
- [Draw.io](https://www.drawio.com/) - 开源绘图工具
- [Draw.io MCP Server](https://github.com/lgazo/drawio-mcp-server) - Model Context Protocol 服务器

## 使用方法

1. 在浏览器中打开 Draw.io
2. 安装 Draw.io MCP Browser Extension
3. 在 Kiro 中配置 drawio-mcp-server
4. 通过对话描述架构需求，让 Kiro 自动生成图表
